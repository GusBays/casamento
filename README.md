# Gustavo & Ana

Site de casamento com home, lista de presentes e checkout Pix.

## Stack

- Next.js 16 App Router
- TypeScript
- Tailwind CSS 4
- shadcn/ui
- Supabase
- Vercel

## Local

```bash
pnpm install
cp .env.example .env.local
pnpm dev
```
